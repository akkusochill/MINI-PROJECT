# 🌍 Data-Driven AQI Trend Analysis

A Python-based data analytics project that explores Air Quality Index (AQI) trends using data visualization and comparative analysis. The project identifies yearly, seasonal, monthly, and daily pollution patterns to generate meaningful environmental insights.

## 🎯 Features

- AQI data preprocessing and cleaning
- Exploratory Data Analysis (EDA)
- Yearly and seasonal trend analysis
- Monthly and daily AQI comparisons
- Data visualization using charts and heatmaps

## 🛠️ Tech Stack

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook
